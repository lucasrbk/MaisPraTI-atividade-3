HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar páginas web. Ele permite que os desenvolvedores estruturam o conteúdo da web usando uma série de elementos e tags. Cada elemento HTML tem um propósito específico e pode ser estilizado usando CSS (Cascading Style Sheets) para criar uma aparência visualmente atraente.

CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Ele permite que os desenvolvedores controlem o layout, as cores, as fontes e outros aspectos visuais de uma página web. Com CSS, é possível criar designs responsivos que se adaptam a diferentes tamanhos de tela e dispositivos.
